# KindPath / KindEarth — Tool Overview

Tools are grouped by posture:
- **Inward-facing** (analysis, pilots, stewardship)
- **Outward-facing** (education, creativity, public engagement)

This separation is intentional.

## Inward-Facing Tools (Core Platform)

### 1. KindEarth Ecological Evaluation Tool
Assess ecological context, constraints, and regenerative capacity of place.

### 2. KindEarth Social Policy Analyser & Forecasting Tool
Detect drift, stress accumulation, and syntropic vs entropic trajectories over time.

### 3. KindPath Social Impact Consulting Tool
Support ethical pilots and institutional collaboration without extraction.

### 4. KindPath Economic Forecasting Tool
Translate social/ecological/trust signals into economic posture. **In-house pilot only.**

### 5. KindPath Digital Library Tool
Micro-community data sovereignty: ownership, provenance, changelogs, consent-aware sharing.

### 6. KindPath Life-Field Tool
Current reality vs ideal reality mapping for practitioners and pilot participants.

## Outward-Facing Tools (Public & Education)

### 7. Syntropic Farming & Food Security Education Tool
Place-based learning for families: food, land, ecological agency.

### 8. ‘Q’ Audio / Music Analysis & Education Suite
Audio learning + creativity + environmental field recording and sharing.
